# LUXSY

A Vite + React website for the LUXSY digital agency.

## Run in StackBlitz

1. Open the project in StackBlitz.
2. Wait for dependencies to finish installing.
3. Start the dev server with:

   ```bash
   npm run dev
   ```

4. Open the local preview URL shown by Vite.

If the preview is blank, open the StackBlitz terminal and browser console. The app includes a visible runtime error panel so React errors are shown on the page instead of failing silently.

## Local commands

```bash
npm install
npm run dev
npm run build
npm run preview
```
